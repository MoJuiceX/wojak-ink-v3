# Sound Effects

Place sound effect WAV files in this directory.

## Required Files

| File | Duration | Description |
|------|----------|-------------|
| `score.wav` | 0.1-0.2s | Short "pop" or "ding" when scoring points |
| `combo-1.wav` | 0.3s | Rising tone for combo level 1-2 |
| `combo-2.wav` | 0.3s | Higher rising tone for combo level 3-4 |
| `combo-3.wav` | 0.3s | Even higher tone for combo level 5-6 |
| `combo-4.wav` | 0.3s | Higher tone for combo level 7-8 |
| `combo-5.wav` | 0.4s | High triumphant tone for combo level 9-10 |
| `combo-max.wav` | 0.5s | Epic triumphant chord for combo 10+ |
| `high-score.wav` | 1-2s | Fanfare, celebration sound |
| `game-over.wav` | 1s | Soft game over sound (not punishing) |
| `game-start.wav` | 0.5s | Game beginning sound |
| `countdown.wav` | 0.3s | 3-2-1 countdown beep |
| `countdown-go.wav` | 0.5s | "GO!" sound after countdown |
| `button-click.wav` | 0.05s | Subtle button click |
| `achievement.wav` | 1.5s | Achievement unlocked fanfare |
| `currency-earn.wav` | 0.3s | Coin/orange collect sound |
| `level-up.wav` | 1s | Level up celebration |
| `warning.wav` | 0.3s | Low time/lives warning beep |
| `success.wav` | 0.5s | Generic success sound |
| `error.wav` | 0.3s | Generic error/fail sound |
| `vote-whoosh.wav` | 0.3s | Emoji flying |
| `vote-splat.wav` | 0.3s | Donut impact |
| `vote-plop.wav` | 0.3s | Poop impact |
| `vote-rain.wav` | 2-4s | Heatmap rain ambient |

## Free Sound Resources

- [Freesound.org](https://freesound.org) - Search "game score", "combo", "achievement"
- [Mixkit.co](https://mixkit.co/free-sound-effects/game/)
- [Kenney.nl](https://kenney.nl/assets?q=audio) - Game-ready sound packs
- [OpenGameArt.org](https://opengameart.org/art-search-advanced?field_art_type_tid=13)

## Sound Guidelines

- Use WAV format for best quality and low latency
- Normalize volume levels to avoid jarring volume differences
- Game sounds should be punchy and satisfying
- UI sounds should be subtle and non-intrusive
