# Sound Effects

Place sound effect MP3 files in this directory.

## Required Files

| File | Duration | Description |
|------|----------|-------------|
| `score.mp3` | 0.1-0.2s | Short "pop" or "ding" when scoring points |
| `combo-1.mp3` | 0.3s | Rising tone for combo level 1-2 |
| `combo-2.mp3` | 0.3s | Higher rising tone for combo level 3-4 |
| `combo-3.mp3` | 0.3s | Even higher tone for combo level 5-6 |
| `combo-4.mp3` | 0.3s | Higher tone for combo level 7-8 |
| `combo-5.mp3` | 0.4s | High triumphant tone for combo level 9-10 |
| `combo-max.mp3` | 0.5s | Epic triumphant chord for combo 10+ |
| `high-score.mp3` | 1-2s | Fanfare, celebration sound |
| `game-over.mp3` | 1s | Soft game over sound (not punishing) |
| `game-start.mp3` | 0.5s | Game beginning sound |
| `countdown.mp3` | 0.3s | 3-2-1 countdown beep |
| `countdown-go.mp3` | 0.5s | "GO!" sound after countdown |
| `button-click.mp3` | 0.05s | Subtle button click |
| `achievement.mp3` | 1.5s | Achievement unlocked fanfare |
| `currency-earn.mp3` | 0.3s | Coin/orange collect sound |
| `level-up.mp3` | 1s | Level up celebration |
| `warning.mp3` | 0.3s | Low time/lives warning beep |
| `success.mp3` | 0.5s | Generic success sound |
| `error.mp3` | 0.3s | Generic error/fail sound |

## Free Sound Resources

- [Freesound.org](https://freesound.org) - Search "game score", "combo", "achievement"
- [Mixkit.co](https://mixkit.co/free-sound-effects/game/)
- [Kenney.nl](https://kenney.nl/assets?q=audio) - Game-ready sound packs
- [OpenGameArt.org](https://opengameart.org/art-search-advanced?field_art_type_tid=13)

## Sound Guidelines

- Keep files small (< 100KB each)
- Use MP3 format for best compatibility
- Normalize volume levels to avoid jarring volume differences
- Game sounds should be punchy and satisfying
- UI sounds should be subtle and non-intrusive
