## Implementation Notes

### FlappyOrange (Reference Implementation)
- First game fully juiced
- Use as reference for all patterns
- Guide: `docs/FLAPPY-ORANGE-JUICE-IMPLEMENTATION.md`
- Design doc: `docs/FLAPPY-ORANGE-DESIGN-ENHANCEMENTS.md`

### Grid Games (BlockPuzzle, MemoryMatch)
- Focus on tile animations, cascade effects
- Near-miss less applicable
- Combo systems are key differentiator

### Action Games (BrickBreaker, OrangePong, KnifeGame)
- Collision feedback is critical
- Time dilation enhances precision moments
- Screen shake on every impact

### Endless Runners (WojakRunner, OrangeSnake)
- Near-miss is very important
- Progressive difficulty needs audio cues
- Distance milestones need celebrations

### Physics/Merge Games (CitrusDrop, OrangeStack)
- ASMR sound design is critical (juicy squish)
- Freeze frames on merge impacts
- Danger zone warning system
- Merge anticipation glow
- Physics bump sounds
- Guide: `docs/games/citrus-drop-juice.md`
