## [0.1.0] - 2025-01-06

### Added

**Generator**
- Layer-based Wojak creation (Background, Base, Clothes, Eye, Head, Mouth)
- Live preview with layer compositing
- Layer-specific pop animations when selecting traits
- Favorites system with localStorage persistence
- "Open Image" feature for saving to Photos (long-press support)
- Random shuffle button
- Multi-select for Mouth extras (cig, cigar, joint, etc.)
- Astronaut blocks conflicting layers (Head, Eye, Mouth)

**Treasury**
- Live wallet balance display (XCH + CAT tokens)
- Interactive crypto bubbles visualization
- NFT collection gallery with modal viewer
- Pull-to-refresh
- Treasury info video easter egg
- Cached data for instant loading

**Gallery**
- Browse all 4200 NFTs by base character (14 bases)
- Interactive trait explorer
- Tap zones to cycle through different traits
- Random shuffle within base

**BigPulp's Take**
- NFT ID search (1-4200)
- Rarity analysis display (rank, tier, percentile)
- BigPulp sentence generation
- S-tier traits, rare combos, named combos display

**Orange Game**
- Tap-to-catch gameplay
- 30-second rounds
- Combo multiplier system (up to 10x)
- High score persistence

**Core**
- Ionic React mobile-first framework
- Tab-based navigation
- Boot sequence with typing animation
- Responsive design (375px+)
